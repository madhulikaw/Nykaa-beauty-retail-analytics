# Nykaa-Inspired Beauty Retail Analytics Dashboard

**Status:** In Progress

## Overview
An end-to-end analytics project inspired by Nykaa (Indian beauty e-commerce),
combining a real product dataset with simulated transactional data to build
a business intelligence dashboard.

## Tech Stack
- SQL (data modeling & querying)
- Python (Pandas, NumPy, Matplotlib — cleaning & EDA)
- Power BI (dashboard & visualization)

## Project Structure
- `data/raw` — original Kaggle dataset (untouched)
- `data/processed` — cleaned data & synthetic transactions
- `sql/` — schema and analysis queries
- `notebooks/` — Python cleaning/EDA notebooks
- `powerbi/` — Power BI report file

## Data Source
[Nykaa Popular Brands Cosmetics & Beauty Products — Kaggle](https://www.kaggle.com/datasets/jithinanievarghese/nykaa-popular-brands-cosmetics-beauty-products)
